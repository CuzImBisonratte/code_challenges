### Fizzbuzz
- Added 23.06.2022
- What to do:
    1. Count from 0 to X
    2. If X is am multiple of 3, output "Fizz"
    3. If X is am multiple of 5, output "Buzz"
    4. If X is am multiple of 3 AND 5, output "Fizzbuzz"
    5. iF it's not a multiple of 3 OR 5, just output the number
